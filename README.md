# git-project

**Опис:**  
Це лендінг, створений за макетом із Figma. Проєкт розроблено для практики HTML/CSS та адаптивної верстки. Сайт адаптований під мобільні пристрої, реалізовано меню-бургер.

---

## Технології:
- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- JavaScript (меню-бургер)

---

## Адаптивність:
- Мобільні екрани  
- Планшети  
- Десктопи

---

## Функціонал:
- Адаптивне меню-бургер  
- Макет повністю зверстано по Figma  

---

## Мета проєкту:
Покращити навички верстки за Figma-макетом, потренуватися в роботі з адаптивом і JavaScript.
